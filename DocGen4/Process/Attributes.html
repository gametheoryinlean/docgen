    class DocGen4.ValueAttr (attrKind : Type → Type) :
    Type 1

    Captures the notion of a value based attributes, attrKind is things like EnumAttributes.

    • getValue {α : Type} [Inhabited α] [ToString α] : attrKind α → Lean.Environment → Lean.Name → Option String

      Given a certain value based attribute, an Environment and the Name of a declaration returns the value of the attribute on this declaration if present.
